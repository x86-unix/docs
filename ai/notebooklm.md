# Google NotebookLM

## 概要

NotebookLM は、Google が提供するAIリサーチアシスタントです。自分のドキュメントをソースとしてアップロードし、その内容に基づいた質問応答・要約・コンテンツ生成ができます。

## 使い方

1. [notebooklm.google.com](https://notebooklm.google.com) にアクセス
2. 「新しいノートブック」を作成
3. ソースを追加（PDF、Google Docs、Webサイト、テキスト、YouTube動画など）
4. ソースに基づいて質問・指示する

## 対応ソース

| ソース種類 | 説明 |
|---|---|
| Google ドキュメント | Google Drive から直接読み込み |
| PDF | アップロードまたはリンク |
| Web サイト | URL を指定して読み込み |
| テキスト | コピー＆ペーストで直接入力 |
| YouTube 動画 | 字幕付き動画から内容を取得 |
| Google スライド | プレゼン資料を読み込み |

## 主な機能

### ソースベースの質問応答

アップロードした資料の内容に限定して回答するため、ハルシネーション（でっち上げ）が起きにくい。回答には必ずソースの引用が付く。

### Audio Overview（音声概要）

ソースの内容を2人のホストが議論する形式のポッドキャスト風音声を自動生成する。

```
使い方:
1. ノートブックを開く
2. 「Audio Overview」をクリック
3. 生成を待つ（数分かかる）
4. 再生またはダウンロード
```

**カスタマイズ:**
- 対象ソースを選択して特定トピックに絞れる
- 「Customize」で聴衆レベル（初心者向け等）を指示可能

### ノート機能

- AI の回答を「ノート」として保存
- 複数ノートを選択して統合・整理
- ノートから新しいコンテンツ（FAQ、ブリーフィング等）を生成

### 自動生成コンテンツ

ソースを追加すると以下が自動提案される：

- **要約** — ソース全体のサマリー
- **FAQ** — よくある質問と回答
- **ブリーフィング** — 要点まとめドキュメント
- **タイムライン** — 時系列整理
- **目次** — 構造化された概要

## 活用例

| シーン | 使い方 |
|---|---|
| 論文リサーチ | 複数論文をアップロード → 横断的に質問 |
| 勉強会準備 | 資料を投入 → Audio Overview で復習用ポッドキャスト生成 |
| 議事録分析 | 議事録PDF → 決定事項とアクションアイテム抽出 |
| 技術書要約 | 本のPDF → 章ごとの要点まとめ |
| 動画学習 | YouTube URL → 内容の質問応答 |

## ポイント

- **ソース限定回答** — アップロードした資料以外の情報は使わない（信頼性が高い）
- **引用付き** — 全ての回答にソースの該当箇所が表示される
- **無料** — Google アカウントがあれば利用可能
- **1ノートブックあたり最大50ソース**
- **共有可能** — 他のユーザーとノートブックを共有できる

## 制限事項

- ソースに含まれない情報には回答できない
- 画像内のテキスト認識は限定的
- 日本語の Audio Overview の品質は英語に比べて発展途上

# Google NotebookLM

## 概要

NotebookLM は、Google が提供するAIリサーチアシスタントです。自分のドキュメントをソースとしてアップロードし、その内容に基づいた質問応答・要約・コンテンツ生成ができます。

## RAG（検索拡張生成）とは

NotebookLM は **RAG（Retrieval-Augmented Generation）** という技術で動いています。

### RAG の仕組み

```
通常のLLM:
  ユーザーの質問 → LLM → 学習データから回答（古い・不正確な場合あり）

RAG:
  ユーザーの質問 → ①関連ドキュメントを検索 → ②検索結果+質問をLLMに渡す → 回答
```

| 比較 | 通常のLLM | RAG（NotebookLM） |
|---|---|---|
| 情報源 | 学習時のデータ | アップロードしたドキュメント |
| 正確性 | ハルシネーションあり | ソース限定で信頼性高い |
| 鮮度 | 学習時点で固定 | 最新資料をアップロード可能 |
| 引用 | なし | 回答に該当箇所の引用付き |
| カスタマイズ | 難しい | ソースを変えるだけ |

### なぜ RAG が重要か

- LLM 単体では「知らないことを自信満々に答える」問題がある
- RAG は「自分の資料」に基づいて答えるので、嘘が起きにくい
- 企業の社内ドキュメント検索、カスタマーサポート等で広く利用されている技術

## 使い方

1. [notebooklm.google.com](https://notebooklm.google.com) にアクセス
2. 「新しいノートブック」を作成
3. ソースを追加（PDF、Google Docs、Webサイト、テキスト、YouTube動画など）
4. ソースに基づいて質問・指示する

## 対応ソース

| ソース種類 | 説明 |
|---|---|
| Google ドキュメント | Google Drive から直接読み込み |
| PDF | アップロードまたはリンク |
| Web サイト | URL を指定して読み込み |
| テキスト | コピー＆ペーストで直接入力 |
| YouTube 動画 | 字幕付き動画から内容を取得 |
| Google スライド | プレゼン資料を読み込み |

## 主な機能

### ソースベースの質問応答

アップロードした資料の内容に限定して回答するため、ハルシネーション（でっち上げ）が起きにくい。回答には必ずソースの引用が付く。

### Audio Overview（音声概要）

ソースの内容を2人のホストが議論する形式のポッドキャスト風音声を自動生成する。

```
使い方:
1. ノートブックを開く
2. 「Audio Overview」をクリック
3. 生成を待つ（数分かかる）
4. 再生またはダウンロード
```

**カスタマイズ:**
- 対象ソースを選択して特定トピックに絞れる
- 「Customize」で聴衆レベル（初心者向け等）を指示可能

### ノート機能

- AI の回答を「ノート」として保存
- 複数ノートを選択して統合・整理
- ノートから新しいコンテンツ（FAQ、ブリーフィング等）を生成

## NotebookLM Studio

Studio は、ソースの内容からさまざまな形式のアウトプットを自動生成する機能です。

### インフォグラフィック

ソースの要点を視覚的なインフォグラフィックとして生成する。

- データや統計をグラフィカルに表現
- プレゼンや報告書に直接利用可能
- 複雑な情報を一目で理解できる形に変換

### フラッシュカード

ソースの内容から暗記・学習用のフラッシュカードを自動生成する。

- 表面に質問、裏面に回答の形式
- 試験勉強や用語の暗記に最適
- ソースの重要ポイントを自動抽出

### スライド資料

ソースの内容からプレゼンテーション用のスライドを生成する。

- 構造化されたスライド資料を自動作成
- 見出し・箇条書き・要点が整理される
- 勉強会やミーティングの発表資料作成を大幅に効率化

### クイズ

ソースの内容に基づいた理解度確認クイズを生成する。

- 選択式・記述式など複数形式
- 学習後の理解度チェックに利用
- チーム研修や自己学習に最適

### その他の自動生成コンテンツ

| 形式 | 説明 |
|---|---|
| FAQ | よくある質問と回答 |
| ブリーフィング | 要点まとめドキュメント |
| タイムライン | 時系列整理 |
| 目次 | 構造化された概要 |
| スタディガイド | 学習の手引き |

## 活用例

| シーン | 使い方 |
|---|---|
| 論文リサーチ | 複数論文をアップロード → 横断的に質問 |
| 勉強会準備 | 資料を投入 → Studio でスライド＆クイズ生成 |
| 試験対策 | 教科書PDF → フラッシュカード＆クイズで反復学習 |
| 議事録分析 | 議事録PDF → 決定事項とアクションアイテム抽出 |
| 技術書要約 | 本のPDF → 章ごとの要点まとめ |
| 動画学習 | YouTube URL → 内容の質問応答 |
| 報告書作成 | データ資料 → インフォグラフィックで視覚化 |

## ポイント

- **RAG ベース** — アップロードした資料のみを情報源とし、ハルシネーションを抑制
- **引用付き** — 全ての回答にソースの該当箇所が表示される
- **無料** — Google アカウントがあれば利用可能
- **1ノートブックあたり最大50ソース**
- **共有可能** — 他のユーザーとノートブックを共有できる

## 制限事項

- ソースに含まれない情報には回答できない
- 画像内のテキスト認識は限定的
- 日本語の Audio Overview の品質は英語に比べて発展途上
